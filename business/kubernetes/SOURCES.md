## 1. Offres d'infogérance Kubernetes en France

### Numeezy - Clusters Kubernetes dédiés et infogérés
- **URL** : https://www.numeezy.com/tarifs-cluster-kubernetes-dedie-et-infogere/
- **Tarifs relevés** :
  - Managed Kubernetes (RANCH) : 20 € à 119 € HT/mois selon ressources
  - Clusters dédiés K-Series :
    - K-S (5 nœuds max) : 400 € HT/mois
    - K-M (10 nœuds max) : 900 € HT/mois
    - K-L (20 nœuds max) : 2 200 € HT/mois
  - Support DevOps : **80 € HT/heure**
  - Services inclus : Rancher, Grafana, Ingress, Velero

### LecPac-Consulting - Infogérance Kubernetes OVHcloud
- **URL** : https://lecpac-consulting.com/infogerance-services-manages-kubernetes-ovhcloud/
- **Informations** :
  - Instance B2-7 (2 vCPUs, 7 Go RAM) : 22 €/mois
  - Stockage : 0,04 €/Go (standard) à 0,08 €/Go (haute performance)
  - Recommandation : minimum 3 nœuds pour production

### Skyloud - Audit, Migration, Build et Infogérance Kubernetes
- **URL** : https://www.skyloud.fr/en/kubernetes
- **Services** : Audit, Migration, Outsourcing, Infrastructure
- **Tarification** : Sur devis personnalisé

### Alfa-Safety - Expert Kubernetes
- **URL** : https://www.alfa-safety.fr/expertises-techniques/expertise-devops/expert-kubernetes-rancher/
- **Services** : Audit, conseil, hébergement Kubernetes
- **Tarification** : Sur devis

### DRI - Hébergement Kubernetes en France
- **URL** : https://www.dri.fr/hebergement-kubernetes
- **Services** : Clusters dédiés, monodatacenter ou multidatacenter
- **Tarification** : Sur devis

## 3. Cloud Providers - Kubernetes Managé

### Azure Kubernetes Service (AKS)
- **URL** : https://azure.microsoft.com/fr-fr/pricing/details/kubernetes-service/
- **Modèle** : Control plane gratuit, facturation VMs/stockage/réseau

### Amazon EKS
- **URL** : https://aws.amazon.com/eks/pricing/
- **Modèle** : Tarif horaire par cluster selon version Kubernetes

### Google Kubernetes Engine (GKE)
- **URL** : https://cloud.google.com/kubernetes-engine/pricing
- **Modèle** : Frais de gestion control plane + compute

### Oracle Kubernetes Engine (OKE)
- **URL** : https://www.oracle.com/cloud/cloud-native/kubernetes-engine/pricing/
- **Tarif** : $0.10/cluster/heure (max $74.40/mois)

### DigitalOcean Kubernetes
- **URL** : https://www.digitalocean.com/pricing/kubernetes
- **Modèle** : Facturation nœuds uniquement
