## About CKAD Certification

The **CKAD** (Certified Kubernetes Application Developer) certification is issued by the **Linux Foundation** and **CNCF**. It validates skills to design, build, and deploy cloud-native applications on Kubernetes.

### Exam Information

| Item | Details |
|------|---------|
| **Duration** | 2 hours |
| **Format** | Performance-based (command line) |
| **Environment** | Real Kubernetes cluster |
| **Passing score** | 66% minimum |
| **Validity** | 3 years |
| **Exam cost** | $445 USD (or course + exam bundle $645) |
| **Attempts** | 1 attempt + 1 free retake |

## Detailed Program

### Day 1: Application Design and Build (20%)

**Morning (3h30)**
- Define, build, and modify container images
- Jobs and CronJobs for batch processing
- Multi-container design patterns:
  - Sidecar
  - Ambassador
  - Adapter
- **Lab**: Optimized image creation and multi-container patterns

**Afternoon (3h30)**
- Using Volumes
- Persistent Volumes and Persistent Volume Claims
- StorageClasses
- **Lab**: Application with different storage types

### Day 2: Application Deployment (20%)

**Morning (3h30)**
- Deployments: creation and updates
- Advanced deployment strategies
- Rolling updates and rollbacks
- **Lab**: Blue/Green and Canary deployments

**Afternoon (3h30)**
- Helm: Kubernetes package manager
- Creating and using Helm charts
- Kustomize for configuration management
- **Lab**: Packaging an application with Helm

### Day 3: Application Observability and Maintenance (15%) + Environment, Configuration and Security (25%)

**Morning (3h30)**
- Understanding API deprecations
- Probes: Liveness, Readiness, and Startup
- Using container logs
- Debugging and troubleshooting applications
- **Lab**: Diagnosing and resolving application issues

**Afternoon (3h30)**
- ConfigMaps: creation and usage
- Secrets: secure management
- SecurityContexts for Pods and containers
- ServiceAccounts and RBAC
- ResourceQuotas and LimitRanges
- **Lab**: Securing an application

### Day 4: Services and Networking (20%) + Mock Exam

**Morning (3h30)**
- Services: ClusterIP, NodePort, LoadBalancer
- Ingress and Ingress Controllers
- NetworkPolicies for network security
- **Lab**: Complete network configuration for an application

**Afternoon (3h30)**
- **Mock Exam**: Real conditions simulation (2h)
- Group review and feedback
- Exam strategies:
  - Time management
  - Using official documentation
  - Essential commands and aliases
- Q&A and final tips
