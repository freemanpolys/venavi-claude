- [Claude Code in Action](https://anthropic.skilljar.com/claude-code-in-action) - Cours officiel Anthropic couvrant l'architecture, les techniques d'implémentation et l'intégration avec MCP servers et GitHub

- [Documentation officielle - Tutoriels](https://docs.anthropic.com/en/docs/claude-code/tutorials) - Guides pas à pas pour explorer du code, corriger des bugs, refactorer, tester
- [Best Practices Anthropic](https://www.anthropic.com/engineering/claude-code-best-practices) - Bonnes pratiques officielles pour le codage agentique
