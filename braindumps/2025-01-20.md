[10:17] Idée SaaS : Gestionnaire de liens GitHub pour développeurs

Problème actuel :
- Sur Twitter/X, je vois des liens de dépôts GitHub intéressants
- Manuellement : je clique, je clone, je fork
- Quand je suis pas sur ordinateur : je m'envoie sur WhatsApp
- Plus tard je viens fork les repos
- Problème : les forks deviennent obsolètes, il faut les mettre à jour

Solution SaaS envisagée :
- Automatiser tout ce workflow
- Agent qui surveille Twitter et clone/fork automatiquement
- Application connectée à GitHub
- Gestion automatique des mises à jour des forks
- Indexation sémantique (vector search) pour rechercher des librairies

Fonctionnalités principales :
1. Inbound de tous les repos (automatique via agent Twitter)
2. Fork automatique dans le compte GitHub connecté
3. Mises à jour automatiques des forks
4. Indexation sémantique pour recherche intelligente

Exemple de référence : https://saveit.now (web + iPad, macOS)
